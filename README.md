# روبوت قراءة السير الذاتية

هذا مشروع Streamlit يقرأ ملفات السير الذاتية (PDF, DOCX, Excel) ويستخرج عدد سنوات الخبرة باستخدام OpenAI.

## طريقة الاستخدام
1. افتح التطبيق على Streamlit Cloud.
2. أضف OpenAI API Key في Secrets باسم `OPENAI_API_KEY`.
3. ارفع السير الذاتية وابدأ التحليل.

## الملفات
- `app.py` : كود التطبيق.
- `requirements.txt` : المكتبات المطلوبة.
